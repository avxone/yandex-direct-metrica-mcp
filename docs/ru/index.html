yandex-direct-metrica-mcp

MCP‑сервер для Yandex Direct + Metrica + Wordstat + Audience — raw‑данные, HF‑инструменты, join’ы и BI‑дашборды (public + pro).

Язык: Русский · English

Проект не аффилирован с Яндексом и не поддерживается Яндексом. Yandex, Yandex.Direct, Yandex.Metrica, Yandex Wordstat, Yandex Audience — торговые марки соответствующих правообладателей.

Что входит

  • Direct: raw‑отчёты и чтение сущностей + HF пресеты отчётов
  • Metrica: raw‑отчёты + Logs API exports + HF отчёты
  • Wordstat: семантика и спрос (raw + HF рекомендации)
  • Audience: каталог сегментов/пересечения/perf proxy (raw + HF)
  • Joins: Direct vs Metrica по UTM / yclid (best effort)
  • Dashboard (Option 1): HTML + JSON, включая multi‑account дашборды
  • BI Option 2 (PRO): датасеты + инкрементальный sync (NDJSON‑friendly)

Установка (Claude Code)

Public (read‑only, safe‑by‑default):

claude mcp add yandex-direct-metrica-mcp -- \
  docker run --rm -i \
    --env-file /path/to/your/.env \
    -e MCP_ACCOUNTS_FILE=/data/accounts.json \
    -v /path/to/your/state:/data \
    ghcr.io/<owner>/yandex-direct-metrica-mcp:latest

Pro (отдельный артефакт; для платных подписчиков; GHCR‑пакет держите приватным):

claude mcp add yandex-direct-metrica-mcp-pro -- \
  docker run --rm -i \
    --env-file /path/to/your/.env \
    -e MCP_ACCOUNTS_FILE=/data/accounts.json \
    -v /path/to/your/state:/data \
    ghcr.io/<owner>/yandex-direct-metrica-mcp-pro:latest

Совет: для ежедневного использования задавайте date_to как вчера (данные за “сегодня” в Direct/Метрике часто неполные).
