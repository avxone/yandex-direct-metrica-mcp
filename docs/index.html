yandex-direct-metrica-mcp

MCP server for Yandex Direct + Metrica + Wordstat + Audience — raw data, HF tools, joins, and dashboards (public + pro).

This project is not affiliated with, endorsed by, or sponsored by Yandex. Yandex, Yandex.Direct, Yandex.Metrica, Yandex Wordstat, Yandex Audience are trademarks of their respective owners.

What you get

  • Direct: raw reporting and entity reads, plus HF report presets
  • Metrica: raw reports + Logs API exports, plus HF reports
  • Wordstat: keyword intelligence (raw + HF suggestions)
  • Audience: segments catalog/overlaps/perf proxy (raw + HF)
  • Joins: Direct vs Metrica by UTM / yclid (best-effort)
  • Dashboard (Option 1): HTML + JSON, including multi-account dashboards
  • BI Option 2 (PRO): datasets + incremental sync (NDJSON-friendly)

Install (Claude Code)

Public (read-only, safe-by-default):

claude mcp add yandex-direct-metrica-mcp -- \
  docker run --rm -i \
    --env-file /path/to/your/.env \
    -e MCP_ACCOUNTS_FILE=/data/accounts.json \
    -v /path/to/your/state:/data \
    ghcr.io/<owner>/yandex-direct-metrica-mcp:latest

Pro (separate artifact; intended for paid subscribers; keep the GHCR package private):

claude mcp add yandex-direct-metrica-mcp-pro -- \
  docker run --rm -i \
    --env-file /path/to/your/.env \
    -e MCP_ACCOUNTS_FILE=/data/accounts.json \
    -v /path/to/your/state:/data \
    ghcr.io/<owner>/yandex-direct-metrica-mcp-pro:latest

Tip: for daily usage set date_to to yesterday (Direct/Metrica data for “today” is often incomplete).
