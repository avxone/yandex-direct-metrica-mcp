# v2.0.3

Date: 2026-02-05

## Changes

### Direct

- `direct.report`: added safe defaults when parameters are omitted:
  - `DateRangeType="CUSTOM_DATE"` (when dates are provided)
  - `Format="TSV"`
  - `IncludeVAT="YES"`
  - `IncludeDiscount="NO"`

This makes the minimal report call work without extra boilerplate.

### Audience

- Public read-only: removed `audience.lookalikes.*` from the tool surface because the upstream API returns HTTP 404 for these endpoints.
- Runtime behavior: if invoked, the server now returns a clear `NotSupported` error with a hint to use `audience.segments.*` tools instead.
