# v2.0.3

Date: 2026-02-05

## Changes

- Direct: `direct.report` now provides safe defaults when parameters are omitted:
  - `DateRangeType="CUSTOM_DATE"` (when dates are provided)
  - `Format="TSV"`
  - `IncludeVAT="YES"`
  - `IncludeDiscount="NO"`
- Audience (public read-only): removed `audience.lookalikes.*` from the tool surface because the upstream API returns HTTP 404; if invoked, the server returns a clear `NotSupported` error with a hint.

## Notes

- Public vs Pro contract remains unchanged:
  - Public images are safe-by-default and read-only.
  - Pro features remain gated by build edition and runtime flags.
