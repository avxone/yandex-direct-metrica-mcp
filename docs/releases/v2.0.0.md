# v2.0.0

Release date: 2026-02-04

## Highlights

- Pro-only OAuth UX tools:
  - `auth.start`
  - `auth.exchange_code`
  - Enabled via `MCP_AUTH_TOOLS_ENABLED=true` (returns secrets; no storage)
- Pro-only write safety:
  - Two-phase write flow with `write.confirm`
  - Enabled via `MCP_TWO_PHASE_WRITES=true` (TTL via `MCP_CONFIRM_TTL_SECONDS`)
- CLI `auth` improved with hybrid loopback callback (`--flow hybrid`) and optional `--output-env`.

## Notes

- Public read-only tool surface remains stable (see `tests/snapshots/public_tools_v1.json`).
- Streamable HTTP transport and hosted/SaaS image are planned for 3.0.0.
